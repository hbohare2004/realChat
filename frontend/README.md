# Real-Time Chat Application Frontend

This is the frontend for the Real-Time Chat Application built with React, Vite, Tailwind CSS, and Socket.io.

## Features
- Real-time messaging
- Online users list
- Typing indicators
- Dark mode theme
- Responsive design

## Tech Stack
- React 18
- Vite
- Tailwind CSS
- React Router DOM
- Axios
- Socket.io-client
- React Icons
- React Hot Toast

## Setup
1. Run `npm install` to install all dependencies.
2. Ensure you have your backend running.
3. Check the `.env` file for the correct API and Socket endpoints.
4. Run `npm run dev` to start the frontend application.

## Folder Structure
- `src/components`: Reusable UI components
- `src/pages`: Application views/pages
- `src/context`: React contexts for state management
- `src/services`: API and WebSocket configurations
- `src/utils`: Helper functions
